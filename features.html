Features

Everything you need to know about what GitBauble is, and how to use it.

Simply put, GitBauble is an IoT connected toy that dances whenever you program it to. The most common use is to have your GitBauble connected as a webhook on any of your GitHub repositories. However, since GitBauble is developer centered, we made sure to give you the ability to connect it to anything you want.

Additionally, GitBauble also can be a physical representation of your current local computer's git status. Your GitBauble will turn green when your local repo is clean, and has no changes. It will turn red when you have uncommitted changes, and it will turn yellow when you have committed changes that have not yet been pushed.

GitBauble has a beginner friendly API which makes connecting your GitBauble to any of your projects very easy. Simply send an application/json POST request to http://gitbauble.com/dance. The only other step you need is to set a header property in your request as 'X-Hub-Signature' or 'Bauble-Signature' with the 20 digit key that is provided with your GitBauble purchase.
